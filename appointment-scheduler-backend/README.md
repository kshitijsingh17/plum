# Appointment Scheduler Backend

Modular monolith Express.js backend for AI-powered appointment scheduling.

Run locally:

1. Copy `.env.example` to `.env` and set values.
2. npm install
3. npm run dev

Endpoints:
- POST /api/ocr
- POST /api/entity
- POST /api/normalize
- POST /api/guardrail
- POST /api/appointment
